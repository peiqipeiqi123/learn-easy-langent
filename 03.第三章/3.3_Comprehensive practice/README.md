# 网页内容查询助手（Web Query Bot）

## 概述

基于 LangChain 构建的网页内容查询智能体，具备**摘要记忆**和**网页抓取工具调用**能力。用户提供网页 URL，助手自动抓取内容，随后可针对网页内容进行提问。

## 技术栈

- **记忆层**：摘要记忆（Summary Memory），自动压缩对话历史，保留核心信息
- **工具层**：自定义 `@tool` 抓取网页，基于 `requests` + `BeautifulSoup`
- **安全层**：内网地址拦截（SSRF 防护）、URL 协议白名单、模型行为约束
- **模型层**：DeepSeek Chat（兼容 OpenAI 接口）

## 安全限制

本 Demo 内置了以下安全机制：

1. **内网地址拦截**：禁止访问 `localhost`、`127.x.x.x`、`192.168.x.x`、`10.x.x.x` 等内网地址
2. **协议白名单**：仅允许 `http` / `https` 协议
3. **模型行为约束**：系统提示词限制 LLM 仅抓取用户明确提供的 URL
4. **请求超时与错误捕获**：防止死连接，优雅处理异常

## 使用方式

```bash
# 安装依赖
uv add requests beautifulsoup4

# 运行
cd 03.第三章/3.3_Comprehensive practice
uv run python 3.3.3_web_query_bot.py
```

启动后：

```
请输入您想提问的信息：https://zh.wikipedia.org/wiki/Python
[调用工具: fetch_webpage]
[工具返回: ...]
助手：已为您抓取网页内容，可以开始提问。

请输入您想提问的信息：Python 有哪些主要特性？
助手：根据网页内容，Python 的主要特性包括...
```

输入 `q` 退出程序。
